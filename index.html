# the dilemma

Is P1 true or false?

(P1) This sentence is false.

# for framing/embedding a Star Trek video clip about the liar's paradox # a brief description of the problem

Here's the issue: if P1 is true then it is true that it is false. And if P1 is false then it is false that it is true. So if P1 is false, then it is true. And if P1 is false, then it is true.

This is quite the tricky dilemma. How do we deal with it? I will discuss the various options below.

# the options

The options

1. Who cares?

To dismiss all this as a trivial issue with no practical significance or value, and to carry on about one's day. (This is probably the most reasonable response).

2.P1 is meaningless

To say that P1 is meaningless. It is like saying 'the concept of a cumulonimbus cloud is happily married to the colour green' (or something). In other words: it's just gibberish hidden behind a grammatically correct polish (so the argument goes). The problem with this view is that there's no obvious principled reason why P1 should be meaningless. For example, it's clear that we can meaningfully talk about the truth of various sentences. For example, consider the following two sentences:

  1. "The opening statement of his court testemony was a bold-face lie."
  2. "When you said 'Bob's puppy is cute' you were speaking the truth."
or respond to sentences by saying "ain't that the truth", and so on). Self-referential sentences can also be meaningful (for example "this sentence is spoken in English", or "this sentence has 22 letters" are both meaningful"). So why is it that when we put these two things together (the truth of a sentence + self reference) suddely the sentence that results from that is not meaningful? Seems like a cop-out.

. There is a hole in P1 and all its truth value is leaking out

One option is to say that there is a so-called 'truth value gap' in P1, so that the sentence is neither true nor false (even if it is not meaningless). But this runs into the same issues as before: if sentences about the truth of sentences have no truth value gap, and self-referential sentences have no truth value gap, then why does a gap suddenly appear when you put these two things together? (Answer: it doesn't. Because this option is, like option 4, merely a cop-out for the weak willed, to scared to face the truth.)
